# MDrive 版本历史

## v1.5.5

- 优化挂载云盘到本地磁盘支持。
- 优化文档和工作流。
- 新增支持自动发布到 GitHub Packages

> 版本说明：如果您是普通用户，请使用独立版本，如果您是开发人员，并且已安装 .NET 依赖，则使用框架依赖版本。

- `MDrvie-SelfContained-x64.zip` 独立版本
- `MDrvie-x64.zip` 框架依赖版本（不包含 .NET 框架）

## v1.5.0

- 增加挂载云盘到本地磁盘支持。
- MDrive 向阿里云产品申请了新的产品线，MDrive 将独立运行，不再依赖 Duplicati。

> 版本说明：如果您是普通用户，请使用独立版本，如果您是开发人员，并且已安装 .NET 依赖，则使用框架依赖版本。

- `MDrvie-SelfContained-x64.zip` 独立版本
- `MDrvie-x64.zip` 框架依赖版本（不包含 .NET 框架）

## v1.1.0

Docker 版本发布，兼容 Windows 和 Linux。
